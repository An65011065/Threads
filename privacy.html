Data Collection and Usage

What We Collect

LINKx collects the following data to create interactive browsing visualizations:

  • URLs and Domains: Website addresses and domain names you visit
  • Tab Relationships: Which tabs opened other tabs to understand navigation flow
  • Navigation Sequences: The chronological order of pages visited within browser sessions
  • Timestamps: When pages are visited, started, and ended for accurate timing

How We Use Your Data

Your browsing data is used exclusively to:

  • Generate directed network graph showing your browsing patterns
  • Calculate browsing metrics like session focus, domain diversity, and navigation efficiency
  • Display session summaries and recent activity in the extension popup

Data Storage and Security

100% Local Storage

Your data never leaves your device.

All graph calculations, session analysis, and metrics are computed entirely on your device.

You have full control to view all collected data through the interactive graph visualization or by exporting the data using the ".csv" button, pause tracking using the "Pause" button, instantly delete all stored data using the "Clear" button, or rely on automatic data expiry after 24 hours without any action needed. All data is automatically removed when you uninstall the extension.

Data is stored using Chrome's secure storage API. with built-in encryption, no external network communication occurs, and the extension follows Chrome Web Store security requirements with automatic data expiration for privacy protection.

Permissions Explained

Required Permissions

"tabs" - Monitor tab creation, navigation, activation, and closure to track browsing sessions and tab relationships
"storage" - Store your browsing data locally on your device using Chrome's secure storage API
"webNavigation" - Track page navigation events to calculate accurate dwell times and build navigation sequences
